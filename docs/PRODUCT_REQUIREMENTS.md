# Product Requirements

## Cortex Enterprise - AI Data Trust & Insights Platform

### 4 Core Screens

1. **Dataset Upload** - Upload CSV/Excel files with preview
2. **Trust Engine** - Data quality analysis and scoring
3. **Insight Copilot** - AI-powered insights and recommendations
4. **Data Catalog** - Searchable metadata catalog

### User Stories
- As a data analyst, I want to upload datasets and get quality scores
- As a data analyst, I want AI-generated insights from my data
- As a team member, I want to search the data catalog

### MVP Scope (10 Weeks)
- Week 1-2: Backend foundation (FastAPI + PostgreSQL)
- Week 3-4: Dataset upload + storage
- Week 5-6: Trust Engine (quality metrics)
- Week 7-8: Insight Copilot (AI integration)
- Week 9-10: Data Catalog + UI polish
