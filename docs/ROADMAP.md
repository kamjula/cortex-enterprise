# Roadmap

## Cortex Enterprise - 10-Week Development Plan

### Phase 0 - Planning (Week 0) ✅
- Project structure setup
- Documentation
- GitHub repo

### Phase 1 - Backend Foundation (Week 1-2)
- FastAPI app scaffold
- PostgreSQL setup
- SQLAlchemy models
- User authentication (JWT)
- Health check endpoint

### Phase 2 - Dataset Upload (Week 3-4)
- File upload endpoint (CSV, Excel)
- DuckDB integration
- Dataset preview API
- Storage management

### Phase 3 - Trust Engine (Week 5-6)
- Data quality metrics
- Great Expectations integration
- Quality scoring algorithm
- Quality report generation

### Phase 4 - Insight Copilot (Week 7-8)
- AI/LLM integration
- Insight generation
- Natural language queries
- Insight history

### Phase 5 - Data Catalog + UI (Week 9-10)
- Metadata catalog
- Search functionality
- React frontend build
- Docker deployment

**Target: Production-ready MVP in 10 weeks**
